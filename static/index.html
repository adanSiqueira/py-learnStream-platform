About

LearnStream is a modular, production-grade backend platform designed for online learning and secure video delivery. It demonstrates a scalable system architecture that combines hybrid database patterns (SQL + NoSQL), event-driven communication, secure JWT-based authentication, role-based access control, and asynchronous webhooks.

Database Architecture

The platform implements a hybrid PACELC-based architecture, combining the strengths of CP (Consistency-Partition) and AP (Availability-Partition) systems.

PostgreSQL (CP/EC)

Consistency-first for critical operations

  • users
  • enrollments
  • refresh_tokens
  • payments (future)

MongoDB (AP/EL)

Availability-first for scalable content

  • courses
  • lessons
  • progress
  • analytics

Security & Authentication

The platform implements JWT-based stateless authentication with hashed refresh tokens for revocation support.

Token Strategy

  • Access Tokens: Short-lived, stateless, verified without DB lookup
  • Refresh Tokens: Long-lived, hashed & stored in PostgreSQL for revocation
